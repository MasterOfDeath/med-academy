# Тестовое задание

## Описание проекта

Это веб-приложение для управления медицинской академией, разработанное на фреймворке Yii2. Проект включает в себя функциональность для работы с авторами, книгами, отчетами и подписками на SMS-уведомления.

## Функциональные возможности

- Управление авторами (CRUD операции)
- Управление книгами (CRUD операции)
- Отчеты о популярных авторах
- Подписка пользователей на получение SMS-уведомлений о новых книгах
- Отправка SMS-уведомлений через внешний API
- Автоматические задания для отправки уведомлений
